# <a name="porting-from-ef6-to-ef-core"></a>Portando de EF6 para EF Core

Devido às mudanças fundamentais no EF Core, não recomendamos tentar mover um aplicativo EF6 para o EF Core, a menos que você tenha um motivo convincente para fazer essa alteração. Você deve entender a mudança de EF6 para EF Core como portabilidade, em vez de uma upgrade.
