> [!TIP]
> <span data-ttu-id="90dbf-119">Gerando imagens nativas para a arquitetura errada é um erro muito comum.</span><span class="sxs-lookup"><span data-stu-id="90dbf-119">Generating native images for the wrong architecture is a very common mistake.</span></span> <span data-ttu-id="90dbf-120">Quando em caso de dúvida, você pode simplesmente gerar imagens nativas para todas as arquiteturas que se aplicam ao sistema operacional instalado no computador.</span><span class="sxs-lookup"><span data-stu-id="90dbf-120">When in doubt you can simply generate native images for all the architectures that apply to the operating system installed in the machine.</span></span>

## <a name="when-to-use-ngenexe"></a><span data-ttu-id="90dbf-122">Quando usar NGen.exe</span><span class="sxs-lookup"><span data-stu-id="90dbf-122">When to use NGen.exe</span></span>  

<span data-ttu-id="90dbf-123">Quando se trata de decidir quais assemblies para gerar imagens nativas para em um aplicativo com base no EF versão 6 ou superior, você deve considerar as seguintes opções:</span><span class="sxs-lookup"><span data-stu-id="90dbf-123">When it comes to decide which assemblies to generate native images for in an application based on EF version 6 or greater, you should consider the following options:</span></span>
